

   ===========================
    Peptideshaker File Reader
   ===========================

   To start PeptideShakerFileReader simply double click the jar file.

   From the command line use 'java -jar PeptideShakerFileReader-X.Y.Z.jar'. 
   (Replace X.Y.Z with the PeptideShakerFileReader version number.)
